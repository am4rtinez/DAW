KeyEvents


Buscar editores

Contiene los eventos tipo:

  • keypress - Suma al contador las teclas presionadas (la tecla retroceso no cuenta).
  • keydown - Suma al contador de vocales si la tecla presionada es vocal.
  • focusin - Cambia de color el fondo del input cuando se hace foco en el.
  • focusout - Cambia de color el fondo del input cuando se sale del foco.

Teclas pulsadas: 0

Vocales pulsadas: 0

Eventos fade, hide y show

Contiene el evento tipo:

  • fadeOut - Clicar sobre la imagen hará que desaparezca.

Contiene el evento tipo:

  • fadeIn - muestra las imagenes ocultas una animación de 3000ms.

Contiene el evento tipo:

  • fadeToggle - muestra/oculta las imagenes con una animación de 3000ms.

Contiene el evento tipo:

  • fadeTo - Hace que las imagenes tengan opacidad o no (animación de 3000ms).

Contiene el evento tipo:

  • hide - Oculta las imagenes.

Contiene el evento tipo:

  • show - Muestra las imagenes.

Contiene el evento tipo:

  • off - Detiene el evento de click sobre las imagenes.

Contiene el evento tipo:

  • one - Muestra un alert y despues deja de funcionar.

Panel mágico

Contiene el evento tipo:

  • slidown - Muestra un panel.
  • slideup / dblclick - Con doble click oculta el panel.

Panel mágico!

Con doble click en "Abracadabra" desaparezco!

Animation


Contiene el evento tipo:

  • animate - Inicia una animacion que rota la imagen.

Contiene el evento tipo:

  • stop - Para la animación si no ha finalizado.
