Pautas de realización

Se entregaran los ejercicios en un solo archivo.rar.

La intención de estos ejercicios es que aprendais CSS desde cero, por este motivo NO se puede utilizar Bootstrap. El CSS ha de ser vuestro en los tres ejercicios. Solo tiene que haber CSS y HTML.

Más adelante ya utilizareis Bootstrap y/u otros frameworks.

Ejercicio 1

Realizad una página web con las caracteristicas siguientes:

  • Mínimo 3 páginas.
  • Se tiene que poner un logotipo en la cabecera.
  • Utilizar barra de navegación o menú desplegable
  • Utilizar combinción de selectores lo máximo posible
  • Utilizar pseudoclases.
  • Utilizar algun Pseudoelemento.
  • Tiene que ser responsive (al menos un móvil).
  • Buenas prácticas y comentarios.
  • Compatible con otros navegadores
  • El CSS en un archivo a parte del HTML.
  • Tener en cuenta el idioma, no puede haber diferentes idiomas. Todo el site tiene que estar en el idioma que hayais elegido.

Ejercicio 2

Crea una animación que te guste. La idea de este ejercicio es practicar lo que hemos visto hasta ahora y que demostreis vuestra creatividad utilizando CSS3.

Ideas:

  • Un cubo con 6 imagenes que gira.
  • Un homonico que camina encima de la barra de menu.
  • Una infografia que tenga movimiento...

Ejercicio 3

  1. Realiza una tabla donde quede reflejado tu horario.

    Intenta poner un efecto 3D para cada celda donde se coloque el cursor (hover).

    Pon cada módulo de un color

  2. Realiza una tabla que tenga las filas alternando de color (tr:nth-child).

Criterios de cualificación

Los tres ejercicios tienen una puntuación sobre la complejidad.

No se puede utilizar Bootstrap (cuando sepais CSS ya lo utilizareis).

Ejercicio 1

Realización de todo lo que se pide (3 puntos).

Complejidad (3 puntos).

La complejidad se definirá y se evaluará por el diseño de las páginas.

  • Es decir, el formato de las imagenes que pongais (claras y que pesen poco).
  • El texto y las cajas bien colocadas, los margenes o los espacios en blanco.
  • La presentación de cada una de las páginas.
  • Tiene que haber información suficiente de manera que se haga scroll en al menos una de las páginas o en algunas cajas donde haya texto.

El tema sera libre. Puede ser una web informativa, de ventas, artística....

Total 6 puntos.

Ejercicio 2

Realización de todo lo que se pide (1 punto).

Complejidad (1 punto).

Total 2 puntos.

Ejercicio 3

Realización (1 punto).

Complejidad (1 punto).

Para la realización de este ejercicio, la complejidad se tendra en cuenta:

  • A la hora de elegir colores, (en este caso los colores serviran para ayudar al lector a que no se puerda en la lectura lineal. Cosa que suele pasar cuando hay muchas linias para leer). Se tiene que pensar que puede ser que este mucho tiempo haciendo una tarea en la que tiene que mirar la tabla.
  • El contenido tiene que ser legible (no se puede quedar una palabra cortada, tamaño de fuente optima...

Total 2 puntos.
