Inicio Animacion Horario

Pautas de realización

Se entregaran los ejercicios en un solo archivo.rar.

La intención de estos ejercicios es que aprendais CSS desde cero, por este motivo NO se puede utilizar Bootstrap. El CSS ha de ser vuestro en los tres ejercicios. Solo tiene que haber CSS y HTML.

Más adelante ya utilizareis Bootstrap y/u otros frameworks.

Ejercicio 1

Realizad una página web con las caracteristicas siguientes:

  • Mínimo 3 páginas.
  • Se tiene que poner un logotipo en la cabecera.
  • Utilizar barra de navegación o menú desplegable
  • Utilizar combinción de selectores lo máximo posible
  • Utilizar pseudoclases.
  • Utilizar algun Pseudoelemento.
  • Tiene que ser responsive (al menos un móvil).
  • Buenas prácticas y comentarios.
  • Compatible con otros navegadores
  • El CSS en un archivo a parte del HTML.
  • Tener en cuenta el idioma, no puede haber diferentes idiomas. Todo el site tiene que estar en el idioma que hayais elegido.

Ejercicio 2

Crea una animación que te guste. La idea de este ejercicio es practicar lo que hemos visto hasta ahora y que demostreis vuestra creatividad utilizando CSS3.

Ideas:

  • Un cubo con 6 imagenes que gira.
  • Un homonico que camina encima de la barra de menu.
  • Una infografia que tenga movimiento...

Ejercicio 3

  1. Realiza una tabla donde quede reflejado tu horario.

    Intenta poner un efecto 3D para cada celda donde se coloque el cursor (hover).

    Pon cada módulo de un color

  2. Realiza una tabla que tenga las filas alternando de color (tr:nth-child).
